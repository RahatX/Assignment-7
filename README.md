# KeenKeeper - Keep Your Friendships Alive

KeenKeeper is a responsive friendship reminder dashboard built for the Programming Hero Assignment 7 brief. It helps users track friends, review who needs attention, log quick interactions, and visualize communication history through a clean KinKeeper-inspired interface.

## Technologies Used

- React
- Vite
- Tailwind CSS
- React Router DOM
- React Toastify
- Recharts
- React Icons

## Core Features

- Figma-aligned navbar with logo, icons, active route styling, and responsive behavior
- Centered home banner with title, subtitle, action button, and four summary cards
- Friend data loaded from `friends.json` with realistic profile content
- Friend card grid with status colors, tags, and navigation to detail pages
- Two-column friend details page with profile info, stats cards, relationship goal card, and quick check-in actions
- Timeline logging for `Call`, `Text`, and `Video` interactions using local storage
- Toast notifications after each quick check-in
- Timeline page with filter options for interaction types
- Friendship Analytics page with a Recharts pie chart
- Custom footer, loading state, 404 page, and SPA redirect support for deployed route reloads
- Responsive layouts for mobile, tablet, and desktop screens
